### 1. [Client Management](./client-management/)
A lightweight CRM system built with Microsoft Power Platform (Power Apps, Power Automate, SharePoint) integrated with Azure AD for managing <1000 clients with segmented communication.

**Features:**
- Client account and contact management
- Monthly segmented email campaigns
- Azure AD group synchronization
- Resource access control
- Email interaction tracking

**Tech Stack:** Power Apps, Power Automate, SharePoint, Azure AD, Outlook
